# **Mdown**
titutulos sao impressos com " # " sendo de 1 a 6 " # ".
**ex** 
# titulo
## titulo
### titulo
#### titulo
##### titulo
###### titulo

para representar alguma palavra ou frase em negrito, basta colocar dois " * * " antes e dois depois da palavra.

**ex**

**exemplo** **frase totalmente em negrito para exemplo**

para colocar em italico o conceito é o mesmo, más coloca-se apensa 1 " * ".

**ex**
*exemplo* *frase completa em italico para exemplo*
